# go-template
The basic building block for Go repositories. This template holds:
- Basic Go application
- Common GitHub workflow
- Pull request template
- Standard .gitignore

## GitHub Workflow
The [Go workflow](.github/workflows/go.yml):
- Runs a linter on your code
- Builds your code
- Tests your code
